# TLA: Three Letter Acronym

This small script is aimed at collecting all 3 letter acronyms 
from an input file `input.txt` into a `database.txt` file

# Usage

```
$ ./run.sh
===================================
            DATABASE
===================================
brb:be right back
btw:by the way
idk:i dont know
lmk:let me know
srt:some random text
```
